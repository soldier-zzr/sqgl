# 在线共用说明

## 1. 启动服务

在当前目录执行：

```powershell
node server.js
```

默认地址：

`http://127.0.0.1:8787`

## 2. 团队内网共用

同一局域网内同事访问：

`http://你的电脑IP:8787`

例如：

`http://192.168.1.20:8787`

## 3. 默认账号

- `admin / 123456`
- `member1 / 123456`
- `member2 / 123456`

## 4. 数据存储

服务端数据保存在：

`db.json`

这是共享数据源，所有同事看到同一份订单。

## 5. 公网访问（后续）

你有云服务器后，拷贝整个目录并运行 `node server.js`，然后用 Nginx 反向代理到 8787 端口即可公网访问。
